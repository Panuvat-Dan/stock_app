# Stock Web Applicaion

This website shows an overview of Fang stock trading open/close/max price

## Description

The technologies contain
- Language : Python
- Framework : Streamlit
- Library : Pandas / Numpy / Yfinance / Datetime / PIL 
- Scraping : Finance Data from Yahoo
